# Coursework-Rostavletov
Курсовая работа
##Описание
АС «ТВ» – это программное обеспечение, автоматизирующее расчёты по основным разделам теории вероятностей. Пользователь вводит исходные данные, выбирает тип задачи, а система выполняет вычисления и выдаёт результат с текстовым пояснением. Место действия программы – графический интерфейс с набором окон, каждое из которых соответствует определённому разделу дисциплины. Исходные данные могут быть целочисленными (количество исходов, число испытаний), вещественными (вероятности, параметры распределений) или табличными (ряды распределения). Результат расчёта отображается в виде числа с заданной точностью и сопровождается пошаговым пояснением применённой формулы.
##Установка
Перейдите в папку с программой. Дважды щёлкните левой кнопкой мыши по файлу AS_TV.exe. На экране появится главное окно программы. При первом запуске выберите нужную точность вычислений (количество знаков после запятой) и нажмите «Сохранить».
##Использование
АС «ТВ» служит инструментом для автоматизации расчётов и обучения теории вероятностей. В учебном процессе она помогает студентам проверять решения и видеть пошаговые пояснения. Преподаватели могут использовать её для быстрой генерации примеров. Специалисты, сталкивающиеся со стохастическими задачами, применяют систему для минимизации вычислительных ошибок (факториалы, интегралы, округления). Программа охватывает ключевые разделы: классическая вероятность, теоремы сложения и умножения, формула полной вероятности и теорема Байеса, схема Бернулли, дискретные и непрерывные случайные величины. Каждый расчёт сопровождается текстовым пояснением с подстановкой чисел в формулу, что выполняет обучающую функцию.
